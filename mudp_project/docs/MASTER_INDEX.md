## Responding to an Incident

### Immediate Response

1. **Triage:**
   - Follow [Crisis Playbook - Immediate Triage](operational/crisis_playbook.md#immediate-triage-checklist)
   - Verify with [Technical Provenance System](technical/provenance_system.md)

2. **Level 1 - Parents:**
   - Use [Parent Notification Templates](callables/templates/TEMPLATE_INDEX.md#32-parent-notification-templates-4-types)
   - Follow [Crisis Playbook Level 1](operational/crisis_playbook.md#level-1-response-parents-only)

3. **Level 2 - School:**
   - Use [School District Notice](callables/templates/TEMPLATE_INDEX.md#33-school-district-notice)
   - Follow [Crisis Playbook Level 2](operational/crisis_playbook.md#level-2-response-school-activation)
   - Reference [Enforcement Tier 2](data/enforcement_tiers.yaml#tier-2)

4. **Level 3 - Platform:**
   - Use [Platform Removal Packet](callables/templates/TEMPLATE_INDEX.md#22-platformapp-removal-packet)
   - Follow [Crisis Playbook Level 3](operational/crisis_playbook.md#level-3-response-platform-removal)
   - Reference [CSAM Law](legal/causes_of_action.md#3-federal-csam-law)

5. **Escalate as Needed:**
   - Follow [Enforcement Tiers](data/enforcement_tiers.yaml)
   - Use appropriate [Legal Causes of Action](legal/causes_of_action.md)
   - Reference [Regulatory Matrix](data/regulatory_matrix.yaml)
