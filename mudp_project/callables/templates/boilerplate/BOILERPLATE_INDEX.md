# Boilerplate Library Index

**Purpose:** Centralized library of mandatory boilerplate blocks for all templates  
**Status:** Core System Component

---

## Overview

All pre-generated templates must include these mandatory boilerplate blocks. This library provides standardized versions for consistent inclusion.

---

## Mandatory Boilerplate Blocks

### 1. Jurisdiction Disclaimer

- [Jurisdiction Disclaimer (Mode A)](jurisdiction_disclaimer_mode_a.md) - For jurisdiction-agnostic templates
- [Jurisdiction Disclaimer (Mode B)](jurisdiction_disclaimer_mode_b.md) - For jurisdiction-specific templates

**Usage:** Insert in all templates. Select Mode A or Mode B based on citation mode.

---

### 2. Non-Waiver of Rights

- [Non-Waiver of Rights](non_waiver_of_rights.md)

**Usage:** Insert in all templates.

---

### 3. Preservation of Evidence Notice

- [Preservation of Evidence Notice](preservation_of_evidence_notice.md)

**Usage:** Insert in all templates. Customize evidence types based on situation.

---

### 4. Good Faith Resolution Statement

- [Good Faith Resolution Statement](good_faith_resolution_statement.md)

**Usage:** Insert in all templates.

---

### 5. Reservation of Escalation Rights

- [Reservation of Escalation Rights](reservation_of_escalation_rights.md)

**Usage:** Insert in all templates. Customize remedies based on applicable legal frameworks.

---

## Integration Requirements

All pre-generated templates must:

1. Include all 5 mandatory boilerplate blocks
2. Select appropriate jurisdiction disclaimer (Mode A or B)
3. Customize placeholders based on situation
4. Maintain consistent formatting
5. Follow drafting rules
