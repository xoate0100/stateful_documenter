### 3. Device Sweep Request

#### 3.1 Scope of Sweep

We request a comprehensive sweep of all devices that may contain the content, including:
- [DEVICE_TYPE_1]
- [DEVICE_TYPE_2]
- [DEVICE_TYPE_3]
- All storage locations
- All backup systems

#### 3.2 Sweep Procedures

**Requested Actions:**
1. Physical inspection of all devices
2. File system search for content
3. Cloud storage review
4. Backup system review
5. Deleted file recovery check (if applicable)
6. Temporary file cleanup
7. Cache and thumbnail cleanup

#### 3.3 Verification Requirements

[Evidence-class-appropriate verification language]

**For E1/E2 (Direct/Platform):**
We request written confirmation with specific details of the sweep results, including device names, locations checked, and verification methods used.

**For E3 (Derived Similarity):**
We request confirmation of the sweep with available details about devices checked and methods used.

**For E4 (Testimonial):**
We request confirmation based on your review of the devices and available information.

**For E5 (Contextual):**
We request confirmation that reasonable steps have been taken to sweep all relevant devices.
