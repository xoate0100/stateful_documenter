## Compliance to Content Connections

### Compliance Rules Apply To
- **All Slides** - Must pass "no sales" validation
- **All Worksheets** - Permission-based language, no urgency
- **All Scripts** - "No sales" disclaimer, choice language
- **All One-Pagers** - Educational only, no sales language
- **All Follow-ups** - Stay educational, permission-based help

### Language Bank Usage
- **Slides** - Use safety lines and permission-based help
- **Scripts** - Use opening/closing language, transitions
- **Worksheets** - Use permission-based framing
- **One-Pagers** - Use educational framing
- **Follow-ups** - Use permission-based help

### Guardrails Enforce
- **Brand Mentions** - Only supported statements about Legacy Lock/SureWealth
- **Legal Content** - Always include disclaimer, encourage attorney consultation
- **Trust & Skepticism** - Acknowledge, reinforce autonomy, provide real tools

## Compliance Validation Flow

### Pre-Generation
1. Review Compliance Rules
2. Check Guardrails
3. Select from Language Bank
4. Verify brand facts are supported

### During Generation
1. Apply Output Specifications
2. Use Language Bank for copy
3. Follow Module content structure
4. Maintain emotional arc

### Post-Generation
1. Validate against Compliance Rules
2. Check for forbidden language
3. Verify disclaimers included
4. Confirm emotional triggers are preferred

## Integration Checklist

When generating any content, verify:

- [ ] Module content aligns with worksheet system
- [ ] Compliance rules are followed
- [ ] Language bank is used appropriately
- [ ] Emotional arc is supported
- [ ] Brand facts are accurate and supported
- [ ] Output specifications are met
- [ ] Cross-references are maintained
