## Usage Guides

### Generating Slides
1. Review [Slide System Rules](compliance/slide_system.md)
2. Follow required slide types (8 types)
3. Apply slide writing rules (4 rules)
4. Use [Language Bank](compliance/language_bank.md) for copy
5. Validate against [Compliance Rules](compliance/compliance_rules.md)

### Generating Worksheets
1. Review [Worksheet Generation Rules](worksheets/generation_rules.md)
2. Select worksheet from [Worksheet Templates](../data/worksheet_templates.yaml)
3. Follow format requirements (1-2 pages, simple language, "10-minute start")
4. Include disclaimer if legal/medical-adjacent
5. Validate against compliance checklist

### Generating Scripts
1. Review [Output Specifications](compliance/output_specifications.md) (Section 11.3)
2. Include "no sales" disclaimer near beginning
3. Use "choice language" (autonomy) throughout
4. Include Q&A transition: "No follow-up requirement from today..."
5. Use [Language Bank](compliance/language_bank.md) for transitions

### Ensuring Compliance
1. Review [Compliance Rules](compliance/compliance_rules.md)
2. Check [Guardrails](compliance/guardrails.md)
3. Use [Language Bank](compliance/language_bank.md) for safe language
4. Validate against compliance checklist
5. Verify no forbidden language patterns

### Understanding Brand Facts
1. Review [Brand & Entity Facts](core_knowledge/brand_facts.md)
2. Check [SureWealth Integration Guidelines](core_knowledge/surewealth_integration.md)
3. Use only supported statements (see [Guardrails](compliance/guardrails.md))
4. Reference [Structured Data](../data/brand_facts.yaml) for exact wording

## Key Principles

1. **Education Only** - No sales pitches, urgency, or scarcity tactics
2. **Autonomy-Respecting** - Permission-based support, no pressure
3. **Emotional Engineering** - Clear before/after states for each module
4. **Compliance-First** - All content must pass "no sales" validation
5. **Atomic Delivery** - Each module has specific emotional outcomes
6. **AI-Ready** - Structured for automated content generation

## Quick Reference

### The Only CTA Allowed
**"Your next step is to complete one worksheet this week. If you get stuck and want help, we're available—no obligation."**

### Required Disclaimers
- **Legal Content:** "This is educational, not legal advice."
- **Opening:** "Today is education only—no sales pitch."

### Critical Language Lines
- **Safety:** "No one is behind—most families don't have this organized."
- **Trust Funding:** "A trust that isn't properly funded may not work as intended."
- **Autonomy:** "You can do this yourself. Help is optional."
