## Overview

The Language Bank provides pre-approved copy blocks that can be used in slides, scripts, worksheets, and other content. These blocks are compliance-safe and support the required emotional arc.

---

## 10.1 Safety Lines (Use Early and Often)

These lines create safety and reduce skepticism. Use them early in the seminar and reinforce throughout.

### "No one is behind—most families don't have this organized."

**When to Use:**
- Opening/intro
- When addressing perfectionism
- When people feel behind
- Module F (Objections)

**Purpose:** Normalizes the situation, reduces shame, creates permission

**Emotional Effect:** Relief, permission

---

### "Today is educational. Take what's useful; ignore what isn't."

**When to Use:**
- Opening/intro (must use)
- Throughout seminar (reinforce)
- When addressing skepticism
- Closing

**Purpose:** Sets expectations, reduces suspicion, creates safety

**Emotional Effect:** Safety, autonomy

---

### "You can do this yourself. Help is optional."

**When to Use:**
- Opening/intro (must use)
- Module D (DIY System)
- Throughout seminar (reinforce)
- Closing

**Purpose:** Reinforces autonomy, reduces pressure, creates permission

**Emotional Effect:** Agency, competence

## 10.3 Trust Funding Line (Critical)

This is a critical line that must be used when discussing trust funding.

### "A trust that isn't properly funded may not work as intended."

**When to Use:**
- Module B (Estate Planning Basics) - when defining Trust
- Module C (Pitfalls) - when discussing unfunded trust risk
- Worksheet 6 (Trust Funding Awareness)

## Additional Language Blocks

### Opening/Intro Language

**"Welcome. Today is education only—no sales pitch. You'll leave with clear next steps and worksheets. You can do this yourself, and help is optional later."**

**Components:**
- Sets expectations (education only)
- Delivers on promise (worksheets)
- Reinforces autonomy (you can do this yourself)
- Permission-based (help is optional)

---

### Module Transitions

**"Let's move to [next topic]. Remember, this is education only. Take what's useful."**

**Purpose:** Reinforces safety and autonomy between modules

---

### Q&A Transition

**"No follow-up requirement from today. If you have questions, I'm happy to answer them. If you want help later, we're available—only if you ask."**

**Purpose:** Reduces suspicion, reinforces no obligation

---

### Closing Language

**"Your next step is to complete one worksheet this week. If you get stuck and want help, we're available—no obligation."**

**Purpose:** Canonical CTA, permission-based, no pressure

## Language Patterns

### Permission-Based Patterns

**"You can..."**
- "You can start with one worksheet"
- "You can do this yourself"
- "You can add more later"

**"If you want..."**
- "If you want help..."
- "If you want a second set of eyes..."
- "If you choose to get help..."

**"Optional..."**
- "Help is optional"
- "Support is optional"
- "This is optional"

---

### Autonomy-Respecting Patterns

**"Your choice..."**
- "The choice is yours"
- "You decide when and if..."
- "You're in control..."

**"No pressure..."**
- "No pressure, no obligation"
- "No urgency"
- "Take your time"

---

### Educational Framing Patterns

**"This is educational..."**
- "This is educational and not legal advice"
- "Today is educational only"
- "This is for educational purposes"

**"You'll learn..."**
- "You'll learn about..."
- "We'll cover..."
- "This will help you understand..."

---

## Forbidden Language Patterns

### ❌ Don't Use

**Urgency:**
- "Time is running out"
- "Limited time offer"
- "Act now"
- "Don't wait"

**Pressure:**
- "You must..."
- "You need to..."
- "Required to..."
- "Critical that you..."

**Fear:**
- "If you die tomorrow..."
- "Your family will suffer if..."
- "You're putting loved ones at risk..."
- "Critical mistake..."

**Shame:**
- "You should have..."
- "Most people have..."
- "You're behind if..."
- "Why haven't you..."

**Sales:**
- "Book your consultation"
- "Schedule now"
- "Limited spots"
- "Special offer"

## Usage Guidelines

### When to Use Language Bank

- **Slides:** Use safety lines and permission-based help
- **Scripts:** Use opening/closing language, transitions
- **Worksheets:** Use permission-based framing
- **One-Pagers:** Use educational framing
- **Follow-ups:** Use permission-based help

### How to Use

1. **Copy directly** - These blocks are ready to use
2. **Adapt slightly** - Can modify for context, but maintain compliance
3. **Combine** - Can combine blocks for longer content
4. **Verify** - Always check against compliance rules

### Quality Check

Before using any language:
- [ ] Is it permission-based?
- [ ] Does it avoid urgency/pressure?
- [ ] Does it support the emotional arc?
- [ ] Is it compliant with "no sales" rules?
- [ ] Does it respect autonomy?
