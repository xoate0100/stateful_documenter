## Generation Rules Summary

### Universal Requirements
- **Length:** 1-2 pages maximum
- **Language:** Simple, plain English
- **Whitespace:** Large whitespace
- **Checkboxes:** Use checkboxes for easy completion
- **Framing:** "10-minute start" or "10-minute completion"

### Required Elements
- Title (clear, descriptive)
- "10-Minute Start" framing
- Purpose statement
- Content elements (checklists, blanks, tables, prompts)
- Disclaimer (if legal/medical-adjacent)
- Emotional before/after labels (optional)

### Language Guidelines
**Do Use:**
- "You can start with..."
- "10-minute start"
- "Choose one..."
- "Permission to..."

**Don't Use:**
- "You must..."
- "Urgent..."
- "Time is running out..."
- Pressure or urgency language
