## Module Summary

### Module A: Legacy Definition (5 min)
- **Purpose:** Establish that legacy planning is a process, not an event
- **Emotional Shift:** Vague anxiety → Conceptual clarity + safety
- **Key Line:** "Legacy planning is not a legal event; it's a process."

### Module B: Estate Planning Basics (15 min)
- **Purpose:** Teach neutral definitions of core documents
- **Emotional Shift:** Overwhelm, confusion → Simplification; "this is manageable"
- **Content:** Will, Trust, Financial POA, Healthcare Directive

### Module C: Pitfalls (10 min)
- **Purpose:** Teach common pitfalls without fear
- **Emotional Shift:** False confidence → Constructive awareness + motivation
- **Content:** Unfunded trust, beneficiary mismatches, outdated docs, digital assets

### Module D: DIY System + Worksheets (15 min)
- **Purpose:** Walk through worksheet system
- **Emotional Shift:** Paralysis → Competence + momentum
- **Content:** 5 worksheets, organization options, "10-minute start" framing

### Module E: Family & Values Legacy (5 min)
- **Purpose:** Add warmth and meaning to the process
- **Emotional Shift:** Sterile "paperwork" vibe → Meaning, connection, calm
- **Content:** Letters, recordings, stories, values transfer

### Module F: Objections (7 min)
- **Purpose:** Reframe common resistance
- **Emotional Shift:** Avoidance → Permission + self-efficacy
- **Content:** "No time", "Too complex", "Not enough assets", "I'll do later"

**Total Content Time:** ~57 minutes (plus intro and closing)
