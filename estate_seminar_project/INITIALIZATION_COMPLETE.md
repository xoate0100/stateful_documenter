## Project Purpose

This project enables an AI LLM to generate any seminar-related asset without additional instructions while maintaining strict "no sales pitch" compliance:

- **Slides** - Presentation decks following canonical structure
- **Worksheets** - 6 canonical worksheets with templates
- **One-Pagers** - 3 standard handouts (A, B, C)
- **Checklists** - Compliance and review checklists
- **Facilitator Scripts** - Presentation scripts with timing
- **Compliant CTAs** - Single canonical CTA only
- **Follow-up Materials** - Educational follow-up content
